VidaWell

Landing Page

Diseñe una landing page para VidaWell, una extensión de Dasana. Esta página sirve como una presentación integral de VidaWell, donde destaco información relevante, incluyendo beneficios, planes disponibles, detalles de contacto y colaboraciones con marcas destacadas. VidaWell se enfoca en permitir a las empresas desarrollar sus propias aplicaciones basadas en Dasana para fortalecer sus relaciones con los clientes fuera de sus establecimientos y mejorar el bienestar y la calidad de vida de sus empleados.

Dasana

Landing Page

En el proceso de rediseño de la aplicación, también se opto por renovar la landing page. Esta decisión se tomó con el propósito de ofrecer a los usuarios la posibilidad de registrarse directamente desde el sitio web, al tiempo que se buscaba proporcionar un diseño más moderno y atractivo a la página.

Dasana

Dashboard

Elaboré un dashboard específicamente para los influencers asociados a Dasana, permitiéndoles acceder a sus métricas, estadísticas, niveles, códigos de referencia y cargar facturas para recibir pagos. Este panel se diseñó principalmente para dispositivos móviles. Sin embargo, se aseguró de que fuera completamente responsivo para garantizar una experiencia óptima.

Electium

App Mobile

Diseñe una aplicación móvil para Electium, con el objetivo de proporcionar a los usuarios la capacidad de mantener un diario virtual personal. Esta aplicación permitía a los usuarios escribir, subir audios y fotos, y establecer dos contactos de confianza durante el proceso de registro. Años después, estos contactos recibirían el contenido almacenado en el diario.
La aplicación también ofrecía otras características importantes. Los usuarios podían explorar blogs de diversos creadores de contenido. Además, la aplicación permitía a los usuarios realizar un seguimiento de los pagos asociados a sus servicios funerarios.
